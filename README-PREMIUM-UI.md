## 🔧 Customization

### Theme Customization
Update CSS variables for instant theme changes:
```css
:root {
  --accent-primary: #your-color;
  --bg-primary: #your-background;
  /* ... other variables */
}
```

### Adding New Pages
1. Add page HTML to `index.html`
2. Add navigation item to sidebar
3. Add page styles to `styles.css`
4. Add page logic to `script.js`

- Follow the established naming conventions
- Maintain the animation and interaction patterns
